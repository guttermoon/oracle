# Nautical vocabulary palette

A resource pool for Starboard copy. Pull from it opportunistically — marketing headlines, blog titles, error states, button microcopy, push notifications, email subject lines, slogans on merch. Do not force every surface to use a nautical term, but reach for one whenever it lands naturally.

This file is the curated vocabulary. The blog post archive at [starboardmanifest.com/blog](https://starboardmanifest.com/blog) is the running voice reference.

| Idiom | Meaning |
|---|---|

| **In the doldrums** | Listless · calm equatorial zone where ships got stuck |
| **Give a wide berth** | Stay away · safe distance between anchored ships |
| **Above board** | Honest, open · items on deck vs hidden below |
| **Scuttlebutt** | Gossip · water barrel where sailors gathered |

| **Cut of her/his jib** | Appearance, character |
| **Even keel** | Stable, balanced |
| **Know the ropes** | Be competent · learning the rigging |
| **Sea legs** | Adjusted to motion · steady on a tossing ship |
| **Dead in the water** | Stalled, no progress · no wind |
| **High and dry** | Stranded · ship beached at low tide |
| **Sailing close to the wind** | Risky behavior |
| **By and large** | Generally, broadly |
| **Flotsam and jetsam** | Odds and ends · floating cargo vs deliberately thrown |

| **A shot across the bow** | A warning |

## How to use this palette in Starboard copy

- **Marketing headlines** — reach for an idiom that fits. *Set your course.* *Steady as she goes.* *Even keel.*
- **Error states** — *Dead in the water — we've lost the connection. Try again.*
- **Success states** — *Anchors aweigh. Your board is saved.*
- **Push notifications** — *Fair winds, Sailor. Time to revisit your board.*
- **Email subject lines** — *Letters from the Helm: Maiden Voyage* · *The doldrums are temporary* · *Following seas this week*
- **Onboarding step labels** — *Cast off · Get underway · Set bearings · Make fast*
- **Empty states** — *Your library is high and dry. Add an image to begin.*
- **Slogans / merch** — *Hold fast.* *Steady on.* *Sea legs.* *Even keel.*

**Restraint matters.** Use one nautical term per surface, not three. If a button just needs to say "Save," let it say "Save." The vocabulary is for moments where the brand voice can breathe — not every UI label.
